About

Why this site exists.

This is where I write down technical lessons before they get rounded off into vague memory.

The writing here is meant to stay practical. Some posts will be proper walkthroughs, but many should only do one job well: capture a useful point clearly.

The main topics are machine learning, large language models, retrieval, ranking systems, infrastructure, and engineering workflow. If something is interesting but still half-formed, it can still belong here as a note.

The publishing standard is intentionally low. If a note explains the problem, the failed intuition, and the lesson I would want next time, it is probably worth posting.
